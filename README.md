# Developer Q & A

How to run the app?

```
 $ neu run
```

How to make a build of React code?

```
 $ cd liteman-rc
 $ yarn run build
```

*Note: If you are on Windows, please run the above command via Git Bash terminal.*

How to make binaries for the latest code?

```
 $ neu build
```
